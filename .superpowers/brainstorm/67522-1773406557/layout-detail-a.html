Workspace Sidebar

  • 宽度 200px，可拖拽，Cmd+B 折叠
  • 活跃 WS：色标实心圆 + 背景高亮 + Tab 列表
  • 非活跃 WS：空心圆，点击激活对应窗口
  • 底部 + New Workspace 入口
  • 通过 WorkspaceManager 单例跨窗口同步
2

File Browser Panel

  • 宽度 260px，可拖拽，Cmd+\ 折叠
  • 锚定 Workspace root_dir
  • Git 状态标注（M/A/D/?）
  • 阶段一嵌入 yazi，阶段二 SwiftUI 原生
  • 顶部过滤框 + 底部 Git 摘要
3

Terminal Area

  • 完全复用 Ghostty 现有 TerminalSplitTreeView
  • Split Tree 递归渲染，支持嵌套分屏
  • SurfaceView (Metal) 零改动
  • 原生 macOS Tab Bar 在标题栏
  • Tab 下划线颜色 = Workspace 色标
4

Status Bar

  • 根目录路径（点击在 Finder 中显示）
  • Git 分支名（点击复制）
  • Git 状态摘要（Modified / Untracked 计数）
  • 运行时版本（Node / Python / Go）
  • Workspace 色标贯穿
